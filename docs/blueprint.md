# **App Name**: NAFON AI Lab

## Core Features:

- AI Project Idea Generation: Generates project ideas based on user prompts, leveraging Gemini AI tool for creative brainstorming and abstracts.
- Smart Project Catalog: Displays available projects with filtering and search capabilities. Also will display recommendations based on popularity or related work
- Order Management: Handles custom orders, integrates project information from form to backend. Integrates all information provided in forms
- Dynamic Content Loading: Uses Next.js for efficient content rendering with AI-generated images. Utilizes computer vision to determine what would be suitable and tasteful

## Style Guidelines:

- Primary color: Forest green (#228B22), evoking nature, growth, and innovation, used for key interactive elements and branding.
- Background color: Soft sage (#DEE7D7), a muted green providing a clean, organic backdrop that enhances readability.
- Accent color: Light olive (#A8B85B), a subtle earth tone used as a secondary highlight for interactive states and non-essential details.
- Font pairing: 'Alegreya', a serif font for headings and titles giving a feel of elegance and tradition. 'Inter', a sans-serif font for body copy allowing modern clear communication.
- Note: currently only Google Fonts are supported.
- Use 'Material Symbols Outlined' for a consistent, modern look.
- Clean, minimalist design to ensure focus on project details. Card-based layouts will be used for presenting individual project items
